About This Project

This Phishing Website Detection System is a security-based web application designed to identify malicious and fraudulent websites.

Objectives

  • Detect phishing and unsafe websites
  • Protect users from online fraud
  • Provide a simple and secure interface

How It Works

Users enter a website URL. The system analyzes the link and displays whether the website is safe or potentially dangerous.

Technologies Used

  • Python & Flask
  • HTML, CSS
  • Cyber Security Concepts

Use Case

This project can be used for cyber awareness, educational purposes, and security demonstrations.
